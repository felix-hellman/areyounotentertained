## Instructions
Create a folder called credentials
{
  "YOUR_EMAIL": "<your_email>",
  "YOUR_PASSWORD": "<your_password>",
  "TICKET_UUID": "<see_instruction>",

## Running the project
Set the FLASK_APP env variable to the entry class
Run

```
flask run
```
Should start on port 5000
